## Where can I find the ADBC driver for Exasol?

**The ADBC Driver for Exasol is community-supported as part of the [Exasol Labs 🧪](https://github.com/exasol-labs) initiative.**

This repo is used by the [ADBC Driver Foundry](https://adbc-drivers.org) to build [Exasol's](https://exasol.com) ADBC driver and make it installable with [dbc](https://docs.columnar.tech/dbc).

---

📥 To install it with [dbc](https://docs.columnar.tech/dbc), run `dbc install exasol`.

🐛 To report an issue, go to [github.com/exasol-labs/exarrow-rs/issues](https://github.com/exasol-labs/exarrow-rs/issues).

⌨️ To see the source code, go to [github.com/exasol-labs/exarrow-rs](https://github.com/exasol-labs/exarrow-rs).

💬 To ask questions, email hello@adbc-drivers.org or chat with us on the [Columnar Community Slack](https://join.slack.com/t/columnar-community/shared_invite/zt-3gt5cb69i-KRjJj~mjUZv5doVmpcVa4w).
